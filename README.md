# Smartphone Price
É um projeto de Ciência de Dados desenvolvido por Gustavo Rocha(eu). No dia 26 de Fevereiro de 2023. Com o intuito de prever valores de Smartphones pela plataforma 
Flipkart

## Quais bibliotecas usadas?
- Scikit-Learn
- Pandas
- Matplotlib
- Joblib
